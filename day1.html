HTML questions & answers.

1.Difference between div and span ?

A. The div tag is also known as division element.It is a block level element.These are the elements starts from a new line.They are used to divide the content on the page into different blocks. A span tag is also known as an in-line element.This is used in the small sections of the content on the page.span do not start a new line.

2.Difference between class and IDs ?

A. A class name can be used by multiple HTML elements.we can use the same class on multiple elements and multiple classes on the same element.classes are not unique.While an id name must only used by one HTML element within the page.IDs are unique

3.What are self closing tags ?

A. Self closing tags are also known as an empty tags.These tags do not have contents and also cannot have any child.Few examples of self closing tags are image tag,input,br,hr,link tag.,etc.

4.How many types of headings are there ?How to decide which to use where ?

A. HTML headings are six levels.They are difined with h1 to h6 tags.h1 tag defines the most important heading or highest level and h6 tag defines the least important heading or lowest level.HTML headings are titles or subtitles that we want to display on a webpage.Headings are used for highlighting important topics.

5.Difference between Block and In-line ?

A. A block-level elements always starts with new line.It can always takes up the full width like streches out to the left and right as far as.While an Inline element does not start on a new line.An inline element only takes up as much width as neccessary or portion of text.

6.Difference between em and strong ?

A. The em tag should be used to reprent stress emphasis.This tag semantically emphasizes on the important word or section of words.while strong tag should be used to indicate strong importance or bold to the content or text.

7.What are attributes ?

A. An HTML attribute is apiece of markup language used to adjust the behavior of an html element.For example,attributes can be used to change the color,size,or functionality of HTML elements.

8.Difference between HTML tags and elements ?

A.HTML tags are starting and ending parts of an HTML element.They begin with < symbol and end with > symbol.While HTML elements are the elements enclose the contents in between the tags.They consist of some kind of structure.It gebnerally consists of a start tag,content and an end tag.

9.Explain br tags ?

A. The br tag is an empty tag which means that it has no end tag or close tag.The br tag inserts a single line break.it is useful for writting addresses or poems.

10.What are semantic tags ?

HTML semantic tags introduce meaning to the web page rather than just presentation.it clearly describes its meaning to the both browser and developer.it is help to screen readers and in code optimization.
