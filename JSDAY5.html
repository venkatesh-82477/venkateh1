Question 2 : Explain Local Scope, Block Scope, Functional Scope and Scope Chain in javascript?

A.local scope :Variables declared within a JavaScript function, become LOCAL to the function.They can only be accessed from within the function.
    
  Block scope : ES6 introduced two important new JavaScript keywords: let and const.
                These two keywords provide Block Scope in JavaScript.
                Variables declared inside a { } block cannot be accessed from outside the block
  
  Functional scope : JavaScript has function scope: Each function creates a new scope.
                     Variables defined inside a function are not accessible (visible) from outside the function.
                     Variables declared with var, let and const are quite similar when declared inside a function.

  Scope chain : Scope Chain means that one variable has a scope (it may be global or local/function or block scope) is used by another variable or
                function having another scope (may be global or local/function or block scope).

Question 6 : Explain passed by value and passed by reference ?

A. Passed by value : It means that every time we assign a value to a variable, a copy of that value is created every single time.
  * Passed by value works with primitive datatype.
  
  Example : let a =45;
            let b = a ;
            b = a + 5;
            console.log(a);
            console.log(b);

  * Passed by reference : When you are creating an object you're given a reference to that object, if 2 variables hold the same reference
                          then changing the object reflects in both variable.
                          
    Example : let obj1 ={
              name :"Ajith",
              age : 23
         }
         let obj2 = obj1;
         obj2.age = 24;
         console.log(obj1);
         console.log(obj2);
